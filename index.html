🔒 Layanan Anti-Phising

Apa Itu Anti-Phising?

Fitur untuk melindungi website dan user dari serangan phishing. Phishing itu penipuan online yang nyamar jadi website asli buat nyuri data login, kartu kredit, dll.

Yang Saya Lakukan:

  • Scan Link Otomatis: Deteksi link keluar yang mencurigakan sebelum user klik
  • SSL & HTTPS: Pastikan website aman dengan sertifikat SSL
  • Warning System: Muncul peringatan kalau user mau keluar ke domain asing
  • Content Security Policy: Blokir script berbahaya dari luar
  • Edukasi User: Halaman edukasi cara kenali website palsu

⚠️ Peringatan Keamanan

Sdev tidak pernah minta password, OTP, atau data kartu kredit via WhatsApp/Email. Kalau ada yang ngaku-ngaku, itu penipuan. Laporkan ke saya.
